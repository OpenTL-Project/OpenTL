Just some Quick notes

you need haxelib SVG, Actuate, OpenFL, and app

simple install commands for them
```
haxelib install svg
haxelib install actuate
haxelib install openfl
haxelib install app
```

Static as of now is meant to be open to hscript in the future to easily modify variables. Keep global settings and vars in there.
